🔹 Overview

This is a personal portfolio website developed using HTML, CSS, and JavaScript, based on Figma design prototypes.
It showcases your skills, projects, certificates, and contact information with a clean, responsive, and interactive layout.

Figma Design: View Prototype

__________________________________________________________________________________________________________________________________________________________________

🔹 Key Features
1️⃣ Sections Overview

Home: Introduction and personal info

About Me: Skills, experience, and education

Portfolio / Projects: Screenshots and links to completed projects

Certificates: Gallery of achievements

Contact: Form or external links

2️⃣ Responsive Design

Works on Desktop, Tablet, and Mobile

Uses CSS Flexbox & Grid for layout

Smooth animations on scroll and hover effects

3️⃣ Interactive Elements

Hover effects on images and buttons

Animated headings and sections

Lightbox/gallery for certificates or project previews

__________________________________________________________________________________________________________________________________________________________________

🔹 Technical Implementation

Languages Used: HTML, CSS, JavaScript

Concepts Used:

Responsive Web Design (RWD)

DOM Manipulation & Event Handling

Figma to Web Conversion

Smooth CSS Animations

Navigate through sections using the menu

No server setup required (Static Website)
